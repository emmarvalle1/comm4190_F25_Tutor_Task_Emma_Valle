# GIT QUICKSTART

# From the repo root in a terminal:
git add -A
git commit -m "Add Tutor Task sessions, prompts, and evaluation"
git push origin main   # or: git push origin master

# If you get “no upstream branch”:
git push --set-upstream origin main

# If permission denied:
# - Ensure you cloned with SSH (git@github.com:...).
# - Check you're in the correct folder (pwd).
